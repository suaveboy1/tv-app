3. Define A Custom Element

Custom Elements

Web Components are a collection of 4 native web APIs: ES Modules, Custom Elements, Shadow DOM, and HTML Templates. You've already used ES modules, which allow you to create JavaScript modules with imports and exports loaded with <script type="module">.

Defining a Custom Element

The Custom Elements specification lets users define their own HTML elements using JavaScript. The names must contain a hyphen (-) to differentiate them from native browser elements.

Example of defining a custom element:

class RatingElement extends HTMLElement {}
customElements.define('rating-element', RatingElement);

Place a <rating-element> in the document body to see how it renders.

Custom Element Lifecycle

Custom Elements come with a set of lifecycle hooks: constructor, connectedCallback, disconnectedCallback, attributeChangedCallback, and adoptedCallback.

Render DOM

Set the element's content when it gets attached to the DOM in the connectedCallback method:

class RatingElement extends HTMLElement {
 constructor() {
   super();
   this.rating = 0;
 }
 connectedCallback() {
   this.innerHTML = \`...HTML and SVG for thumbs up and down...\`;
 }
}
customElements.define('rating-element', RatingElement);

This adds DOM children to <rating-element> to display the current rating with thumbs up and thumbs down buttons.
