2. Getting Set Up & Exploring the Playground

Accessing the Code

Throughout the codelab, there will be links to the Lit playground. The playground is a code sandbox that runs completely in your browser, capable of compiling and running TypeScript and JavaScript files, and resolving imports to node modules.

Example:

// before
import './my-file.js';
import 'lit';

// after
import './my-file.js';
import 'https://unpkg.com/lit?module';

You can complete the entire tutorial in the Lit playground, using these checkpoints as starting points.

Exploring the Lit Playground UI

The file selector tab bar is labeled Section 1, the code editing section as Section 2, the output preview as Section 3, and the preview reload button as Section 4.

  • File selector: Note the plus button...
  • File editor.
  • Code preview.
  • Reload button.
  • Download button.

Use ctrl + s (cmd + s for Mac) to save your progress as a URL, allowing you to reload the page without losing progress or share your code with others.

VS Code Setup (Advanced)

If you're running this codelab entirely in the browser, you can skip this step. The benefits of using this VS Code setup include template type checking and template intellisense & autocompletion.

To get started:

  1. Press the download button.
  2. Extract the tar file's contents into a directory.
  3. Install a dev server that can resolve bare module specifiers (e.g., @web/dev-server).
  4. Run the dev server and open your browser.
  5. For the example package.json, use npm run serve.
