Property Bindings

Update the custom element view when the rating property changes by adding a setter and getter. This enables the view to update when the property is set programmatically.

class RatingElement extends HTMLElement {
  constructor() {
    super();
    this._rating = 0;
  }
  set rating(value) {
    this._rating = value;
    // Update logic
  }
  get rating() {
    return this._rating;
  }
}
customElements.define('rating-element', RatingElement);
